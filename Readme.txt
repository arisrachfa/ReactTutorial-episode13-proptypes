Pada episode 13-proptypes ini membahas tentang penggunaan proptypes package untuk mengkoreksi code kita jika ada
kesalahan dalam memberikan sebuah inputan dengan type data yang berbeda.
kegunaan propTypes ini untuk memberikan kita informasi jika inputan yang kita masukan memiliki type data yang berbeda dengan,
kebutuhan yang dibuat pada sebuah state/( function ), namun propTypes hanya memberikan kita informasi error, tidak
menjadikan program kita error oleh karena itu program akan tetap berjalan.

kenapa Type Safety penting dalam react karena untuk menghindari ketidaksamaan antara type data yang dibutuhkan dengan 
type data yang diinputkan.

untuk menginstal propTypes bisa dengan command : npm install prop-types
